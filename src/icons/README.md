# Icons

Velocity uses **[Remix Icon](https://remixicon.com/)** via [`react-icons`](https://react-icons.github.io/react-icons/icons?name=ri) (`react-icons/ri`).

Import only the icons you need:

```tsx
import { RiHeartLine, RiCloseLine } from "react-icons/ri";
```

`react-icons` is a **peer dependency** — install it alongside `@runswap/velocity`.
